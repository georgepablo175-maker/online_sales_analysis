# Online Sales Analysis

## Descriere proiect
Acest proiect Python simulează un sistem simplu de gestionare a produselor și a unui coș de cumpărături.

## Funcționalități

### Clasa Product
- Stochează informații despre produs (nume, preț, cantitate)
- Afișează informațiile produsului
- Permite actualizarea cantității

### Clasa ProductManager
- Gestionează lista de produse
- Adaugă și elimină produse
- Afișează produsele disponibile
- Calculează valoarea totală a inventarului

### Clasa Cart
- Gestionează produsele din coș
- Adaugă produse în coș
- Afișează conținutul coșului
- Calculează valoarea totală de plată

## Structura proiectului
- product.py
- product_manager.py
- cart.py
- main.py
